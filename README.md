# Mata Moscas — Game em JavaScript

Um mini game simples e divertido feito para **fixar conteúdos estudados em JavaScript**: o objetivo é clicar nos mosquitos antes que eles sumam da tela.

> Projeto criado com foco em prática (HTML + CSS + JS).

---

## Sobre o projeto

Esse projeto nasceu como um exercício prático para treinar:

- Manipulação do **DOM**
- **Eventos** (cliques, interações do usuário)
- **Timers/intervalos** e lógica de tempo de jogo
- **Regras de gameplay** (vitória/derrota, dificuldade, etc.)
- Organização básica de arquivos **HTML / CSS / JavaScript**

---

## Tecnologias usadas

- **HTML**
- **CSS**
- **JavaScript**
- **BootStrap**

---

## Como jogar

1. Abra o jogo no navegador (localmente ou pelo link do GitHub Pages quando estiver publicado).
2. Clique nos mosquitos o mais rápido possível.
3. Se deixar passar… você perde vidas/tempo (dependendo da regra do jogo).
4. link do jogo no github Pages: https://solidusjack.github.io/Mata_moscas_Game/pagina_inicial.html

---

## Rodando localmente

Você pode rodar de forma bem simples:

1. Baixe/clone o repositório:
   ```bash
   git clone https://github.com/SolidusJack/Mata_moscas_Game.git
   ```
2. Abra o arquivo `index.html` no navegador.

Dica: se quiser, também pode usar a extensão **Live Server** no VS Code para facilitar.
